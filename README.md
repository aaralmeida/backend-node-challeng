### <a name=“server”>Servidor<a/> 

Para Iniciar o servidor rode o seguinte comando na raíz.  
`npm start -- --log-level trace --port 3001`

Para executar testes unitários, instale o mocha:
`npm install mocha -g`
E execute o teste:
`npm run unit`
